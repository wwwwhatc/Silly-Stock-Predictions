Tushare API Token：填写在Tushare注册的API Token
<br>
股票代码：填写自己想要查询的股票，格式为：300099.sz
<br>
选择日期：填写自己想要查询股票的日期，格式为：20200101
<br>
预测日期：填写自己想要查询的股票的预测日期，格式为：20200101（注：0.1版本预测时间不宜超过24小时）
